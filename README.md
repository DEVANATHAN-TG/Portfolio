# Devanathan T — Portfolio Website

Personal portfolio website showcasing my work as an **AI/ML Engineer & Full-Stack Developer**.

live Link - https://devanathan-t-portfolio.vercel.app/

## Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Vanilla CSS (dark-mode, minimalist design)
- **Animation:** Framer Motion + custom canvas-based neural background
- **Icons:** Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── App.jsx                         # Root component
├── main.jsx                        # Entry point
├── index.css                       # Global styles & design tokens
├── components/
│   ├── Navbar.jsx                  # Fixed header with mobile hamburger
│   ├── Hero.jsx                    # Hero section
│   ├── InteractiveNeuralBackground.jsx  # Animated neural-net background
│   ├── Skills.jsx                  # Technical skills showcase
│   ├── Projects.jsx                # Project listing
│   ├── Contact.jsx                 # Contact form section
│   └── Footer.jsx                  # Site footer
└── data/
    └── portfolioData.jsx           # Centralised portfolio content
```
